# TKH Data Analytics Course

Welcome! This repo contains everything you need for the TKH Data Analytics course. You'll go from Python fundamentals all the way through real-world data analysis — exploratory data analysis, statistics, visualization, and machine learning concepts.

---

## Repo Structure

```
00_orientation/       ← Start here
working-sessions/     ← Main course notebooks (follow along in class)
labs/                 ← Practice notebooks (complete on your own)
data/                 ← Shared datasets
```

---

## Getting Started

1. **Fork this repo** to your own GitHub account
2. Open your fork and click **Code → Codespaces → Create codespace on main**
3. Wait for the environment to finish building (this takes a few minutes the first time)
4. When prompted about widgets downloading from a third-party site, click **Ok**
5. Open any notebook and select this kernel:
   ```
   Python 3.12.13 — /usr/local/python/current/bin/python
   ```
6. Head to **`00_orientation/`** and start there

---

## End of Session

Before you close your Codespace:

- **Commit and push your work** so nothing is lost
- Go to [github.com/codespaces](https://github.com/codespaces) and **stop your Codespace** — this prevents it from running in the background and using up your free hours
